# Почвоведческая экспертиза

Почвоведческая экспертиза - это исследование почв, определение их состава, свойств, а также установление факта загрязнения и порчи почв.

## Основные направления почвоведческой экспертизы

- Определение состава и свойств почв
- Исследование степени загрязнения
- Оценка плодородия почв
- Установление факта порчи почв
- Анализ соответствия экологическим нормам

## Когда необходима почвоведческая экспертиза

Почвоведческая экспертиза требуется в следующих случаях:

- При спорах по земельным участкам
- При проверке экологической обстановки
- При оценке ущерба от загрязнения
- При оформлении прав на землю
- При оспаривании результатов экологической экспертизы

## Методы проведения экспертизы

Наши специалисты используют современные методы и оборудование для проведения почвоведческой экспертизы:

- Лабораторные исследования
- Химический анализ
- Биологический анализ
- Физический анализ
- Экспертные заключения по почвоведческим вопросам
