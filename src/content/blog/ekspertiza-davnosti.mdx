# Экспертиза давности

Экспертиза давности - это определение времени создания документов, нанесения записей, почерковых элементов и других следов.

## Основные направления экспертизы давности

- Определение времени создания документов
- Установление времени нанесения записей
- Исследование времени появления почерковых элементов
- Оценка времени формирования следов
- Анализ временных характеристик материалов

## Когда необходима экспертиза давности

Экспертиза давности требуется в следующих случаях:

- При спорах по документам
- При оспаривании даты составления документов
- При подозрении на подделку документов
- При установлении хронологии событий
- При расследовании мошенничества

## Методы проведения экспертизы

Наши специалисты используют современные методы и оборудование для проведения экспертизы давности:

- Химический анализ материалов
- Микроскопическое исследование
- Спектральный анализ
- Радиоуглеродный анализ
- Экспертные заключения по временным вопросам
